# Claude Code Toolkit

This repo contains personal customizations for the Claude Code CLI.

## Project Context
- Owner: Johnny (colaboy519 on GitHub)
- Purpose: Track, version, and improve Claude Code CLI configurations
- All scripts target Windows 11 + Git Bash environment

## Conventions
- Scripts in `scripts/` should be Node.js (matching the statusline pattern)
- Keep configs as JSON, document any non-obvious settings
- Test scripts locally before committing
